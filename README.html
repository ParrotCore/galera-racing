Wymagania

Gra jest w pełni stworzona przy użyciu HTML, CSS, oraz JS, nie zawiera w sobie dodatkowych bibliotek, kod jest ogólnodostępny i nie zawiera w sobie żadnego szkodliwego oprogramowania. Jedyne, czego potrzebujesz, to przeglądarki Google Chrome, która jak na razie jako jedyna ją wspiera, również ważnym elementem jest zezwolenie na javascript w ustawieniach twojej przeglądarki, jeśli masz je wyłączone. Wystarczy uruchomić plik index.html i voila, jesteś w grze!
